# TaskFlow - Sistema de Gerenciamento de Tarefas

**Desenvolvedor:** Marcio Costa Barbosa (RA: 190093)
**Disciplina:** Software Engineering - UNIFECAF

## Sobre

Sistema de gerenciamento de tarefas desenvolvido para TechFlow Solutions, permitindo acompanhamento de fluxo de trabalho em tempo real usando metodologias ágeis.

## Tecnologias

- Python 3.9+
- Flask
- Pytest

## Como Executar

```bash
# Instalar dependências
pip install -r requirements.txt

# Executar aplicação
python src/app.py
```

##  Testes

```bash
pytest
```
